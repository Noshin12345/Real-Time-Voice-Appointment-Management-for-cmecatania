<p>It handles appointment stages, start/end times, patient details, treatment info, agent sources, and slot availability — all in an automated pipeline that improves response time, reduces errors, and centralizes appointment management.</p>

<p>Features</p>
<p>Receives appointment requests via webhook</p>
<p>Normalizes input data and validates required fields</p>
<p>Resolves treatment specialties and agenda mappings automatically</p>
<p>Checks availability across multiple agendas and returns optimal slots</p>
<p>Supports scheduling, rescheduling, and cancellations</p>
<p>Generates structured, AI-friendly analytics for every booking operation</p>
<p></p>Fully production-ready for real-time deployment</p>
